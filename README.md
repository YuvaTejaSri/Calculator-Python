# 🧮 Calculator (Python)

## 📌 Overview
A simple command-line calculator built in Python that performs basic arithmetic operations.  
The application allows users to input numbers and select operations through a menu-driven interface.

This project was developed during internship practice to strengthen Python fundamentals and logical problem-solving skills.

---

## ✨ Features
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 💻 Menu-driven CLI interface  
- ⚡ Fast and lightweight  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Concepts Used:** Functions, Conditionals, User Input
